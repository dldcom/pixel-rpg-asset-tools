# Source Assets

This folder stores original images, prompts, manifests, and project-specific source material.

Recommended rule:

- Keep editable or generated originals here.
- Keep game projects focused on final runtime assets.
- Use `output/` for optimized exports generated by scripts.

Example:

```text
source-assets/
  society-4-1-2-2/
    src-assets/
      buildings/
      characters/
      maps/
      tilesets/
```
